Deadset

In your CSS, define the amount of columns that you want an element to span. Put each section of content within an element and call the "row" mixin on it. Class names are arbitrary and used only for examples. The mixins are the magic.

All credit to https://github.com/tylertate/semantic.gs/. I just tweaked it a bit.
